# Video Search Interface

Учебный проект по вёрстке страницы поиска видео.

Цель проекта — отработка навыков построения лейаута, работы с Flexbox и стилизации элементов формы.

## Технологии

- HTML5  
- CSS3  
- Flexbox  
- Псевдоклассы и псевдоэлементы  

## Реализовано

- Центрирование основного контейнера  
- Построение лейаута с помощью Flexbox  
- Стилизация текстового поля ввода  
- Кастомный чекбокс  
- Состояния элементов (:hover, :active, :focus-visible)  
- Ограничение текста с помощью text-overflow  
- Ограничение описания через line-clamp  
- Прокручиваемый список карточек  

## Запуск проекта

1. Клонировать репозиторий  
2. Открыть файл `index.html` в браузере
